# Car Workshop Mini API
Simple backend system for managing customers, vehicles and jobs.

## Setup
npm install
npm run dev

## Features
- Customer management
- Vehicle tracking
- Job service records (basic implementation)

## Notes
- No database used (in-memory only)
- No authentication implemented
- Some features may be incomplete or under development
- Basic validation is implemented in some modules only

## API Structure
Routes are grouped under `/api` prefix.

Example:
- GET /api/customers
- POST /api/jobs

## Developer Notes
- Controller structure is not fully consistent across modules
- Error handling is partially implemented
- This project is intended for testing and learning purposes
